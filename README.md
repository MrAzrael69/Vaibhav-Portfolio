# Vaibhav Chavan - Portfolio

A modern, interactive portfolio website showcasing my work as a Full-Stack and Blockchain Developer.

## Live Demo

[View Portfolio](https://mrazrael69.github.io/Vaibhav-Portfolio/)

## Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first styling via CDN
- **JavaScript** - Vanilla JS for interactions
- **Three.js** - 3D animated background

## Features

- Responsive design that works on all devices
- Interactive 3D crystal animation in the hero section
- Custom cursor with hover effects
- Smooth scroll animations
- Scroll progress indicator
- Konami code easter egg (try it!)

## Project Structure

```
Portfolio/
├── index.html          # Main HTML file
├── css/
│   ├── animations.css  # Keyframe animations
│   └── components.css  # Component styles, cursor
└── js/
    ├── three-scene.js  # 3D background setup
    ├── cursor.js       # Custom cursor logic
    ├── animations.js   # Scroll reveals, typing effect
    └── main.js         # Core functionality
```

## Running Locally

Just open `index.html` in your browser. No build step required.

```bash
# Or use a local server
npx serve .
```

## Sections

1. **Hero** - Name, title, and 3D animated background
2. **About** - Brief introduction
3. **Tech Arsenal** - Skills organized by category with proficiency indicators
4. **Featured Work** - Project showcase
5. **Core Competencies** - Key strengths
6. **Opportunities** - What I'm looking for
7. **Contact** - Get in touch

## Customization

To use this template for your own portfolio:

1. Update personal info in `index.html`
2. Replace project details in the Featured Work section
3. Modify colors in the Tailwind config (inside the `<script>` tag)
4. Update social links in the footer
